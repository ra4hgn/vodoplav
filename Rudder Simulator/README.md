## Rudder Simulator

Прошивка для **ESP32-S3 DevKitC-1**, которая эмулирует устройство управления рулём (Steering Control) в сети NMEA 2000.

### Назначение

Устройство считывает угол поворота с **потенциометра** (аналоговый вход GPIO5) и отправляет данные по **CAN-шине** в виде PGN-сообщения **127245 (Rudder)**. Может использоваться для тестирования и отладки NMEA 2000-совместимых приборов (дисплеев, картплоттеров, автопилотных систем и т.п.).

### Аппаратная конфигурация

| Компонент       | GPIO  | Назначение                        |
|-----------------|-------|-----------------------------------|
| POT (крутилка)  | 5     | Аналоговый вход (ADC, 12 бит)     |
| CAN TX          | 17    | Передача данных по CAN (TWAI)     |
| CAN RX          | 16    | Приём данных по CAN (TWAI)        |
| LED статус      | 15    | Индикация работы устройства       |
| LED активность  | 3     | Кратковременная вспышка при отправке |

### Принцип работы

1. **Измерение**: АЦП считывает напряжение с потенциометра (диапазон 0–4095)
2. **Мэппинг**: Значение АЦП линейно преобразуется в угол поворота руля: **0 → −40°** (лево), **2047 → 0°**, **4095 → +40°** (право)
3. **Отправка**: Каждые **200 мс** формируется и отправляется PGN 127245 (Rudder) с текущим углом в радианах
4. **Индикация**: При каждой отправке данных кратковременно зажигается светодиод активности (GPIO3)

### Технические детали

- **Платформа**: ESP32-S3 (ESP32-S3-DevKitC-1)
- **Фреймворк**: Arduino (через PlatformIO)
- **CAN**: TWAI (Two-Wire Automotive Interface), совместимый с NMEA 2000 Physical Layer
- **Режим NMEA 2000**: SendOnly (только передача)
- **Адрес устройства**: 22 (Source Address)
- **Период отправки**: 200 мс
