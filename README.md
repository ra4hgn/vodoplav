# vodoplav

Монорепозиторий проектов для NMEA 2000 на ESP32-S3.

## Проекты

### [Rudder Simulator](./Rudder%20Simulator/)

Прошивка для **ESP32-S3 DevKitC-1**, которая эмулирует устройство управления рулём (Steering Control) в сети NMEA 2000. Устройство считывает угол поворота с потенциометра и отправляет данные по CAN-шине в виде PGN-сообщения 127245 (Rudder).

[Подробнее →](./Rudder%20Simulator/)
